Detailed Description

A class that calculates replay gain.

See also
http://wiki.hydrogenaudio.org/index.php?title=ReplayGain_specification

To calculate an album's replay gain, create a ReplayGainAnalyzer and all ReplayGainAnalyzer::AnalyzeURL()

CFErrorRef *  error = nullptr

Analyze the given URL's replay gain.

If the URL's sample rate is not natively supported, the replay gain adjustment will be calculated using audio resampled to the sample rate returned by ReplayGainAnalyzer::GetBestReplayGainSampleRateForSampleRate()
