Detailed Description

template<typename T>
class SFB::CFWrapper< T >

A wrapper around a Core Foundation object.

CFWrapper simplifies the use of CFTypes in C++ by wrapping a CF object, ensuring CFRelease will be called when the CFWrapper goes out of scope.
